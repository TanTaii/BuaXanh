FoodSaver Admin Setup

Trang này chỉ dùng 1 lần để cấp quyền Admin

⏳ Đang chờ...

✅ Đã set Admin thành công!

Bạn có thể xóa file setup-admin.html sau khi hoàn tất.

❌ Lỗi:

📋 Hướng dẫn thủ công (nếu cần):

  1. Vào Firebase Console → Authentication
  2. Tìm email quantrifs@gmail.com → Copy UID
  3. Vào Realtime Database
  4. Thêm: users/<UID>/isAdmin = true
  5. Thêm: users/<UID>/role = "admin"
